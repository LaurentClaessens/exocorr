This is the so-called "documentation" of SystemeCorr.sty


This package is intended to manage exercise/corrections in a LaTeX document dedicated to education. You need to create one separated file for each exercise and each correction. (2n file for n exercises).

The package also contains a command dedicated to link to Wikipedia.

Read the example file example.tex to learn how to use.

The script liste_exo.py automatically create the needed files use :
liste_exo.py NameOfMyListOfExercises 1 30
this creates 30*2 files with the labels and everything you need.

Read the description inside liste_exo.py for efficient use.
